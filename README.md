# PintaPojat – Julkisivumaalaus Joensuussa

PintaPojat tekee julkisivumaalauksia, huoltomaalauksia ja pieniä pintaremontteja Joensuussa sekä Joensuun seudulla.

Sivusto:  
https://pintapojat.fi

## Palvelut
- Maalaus
- Talonmaalaus
- Julkisivumaalaus
- Huoltomaalaus
- Hirsipintojen käsittely
- Sokkelimaalaus
- Pienet pintaremontit

## Toiminta-alue

- Joensuu
- Pohjois-Karjala
- lähialueet
